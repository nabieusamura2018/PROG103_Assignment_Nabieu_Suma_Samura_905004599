# Clinic Queue Management System (CQMS)

## Overview
The Clinic Queue Management System (CQMS) is a terminal-based Python application developed for the PROG103 Principles of Structured Programming assignment.

The system is designed to help manage patient queues in a clinic environment in a simple, organized, and efficient manner.

---

## Features
- Register patients
- Store patient records
- View patient queue
- Search patient records
- Serve patients from the queue
- Generate queue summary
- Input validation for accurate data entry
- Menu-driven terminal interface

---

## Programming Concepts Used
This project demonstrates the use of:
- Functions
- Variables and data types
- Lists and dictionaries
- Conditional statements
- Loops
- Input validation
- Structured programming principles
